Github

  1. Go to Github
  2. Make new repository
  3. Copy SSH link
  4. Go to terminal
  5. Type 'git clone (paste)'
  6. Got into folder
  7. Create html file 'touch index.html'
  8. Add file, 'git add index.html'
  9. Commit file with comment, 'git commit -m (COMMENT)
  10. Push file, 'git push'
